# Mentor

Autonomous AI project manager for developers running many parallel Claude Code projects. Watches all your sessions, decides what matters, takes action on its own. Uses your existing Claude subscription — no API key needed.

Status: alpha (v0.1).

## Install (when released)

```bash
curl -fsSL https://mentor.sh/install | sh
```

## Build from source

```bash
go build -o bin/mentor ./cmd/mentor
go build -o bin/mentord ./cmd/mentord
```
